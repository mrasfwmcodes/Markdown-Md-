# heading
Heading atau judul di markdown dapat dibuat dengan tanda pagar `#`.

# heading 1
```md
# heading 1
```
## heading 2
```md
## heading 2
```
### heading 3
```md
### heading 3
```
#### heading 4
```md
#### heading 4
```
##### heading 5
```md
##### heading 5
```
###### heading 6
```md
###### heading 6
```

> Alternatively, for H1 and H2, an underline-ish style:

```md
Alt-H1
======

Alt-H2
------
```
